GitHub Contribution Color Graph
===============================

[![Build Status][travis-image]][travis-url]
[![Dependency Status][gemnasium-image]][gemnasium-url]

Change colors of contribution graph in GitHub.

Setup
-----

```bash
$ npm install
```

Test
----

```bash
$ npm test
```

Build
-----

```bash
$ grunt build
```
